# documentation 

這個專案將會實現多個排程演算法，並包含以下 C 檔案：
schedule_fcfs.c - 實現 FCFS 排程演算法。
schedule_sjf.c - 實現 SJF 排程演算法。
schedule_rr.c - 實現 RR 排程演算法。
schedule_priority.c - 實現優先權排程演算法。
schedule_priority_rr.c - 實現優先權 RR 排程演算法。
